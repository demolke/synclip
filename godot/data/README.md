# Test data assets

## head.glb

Human head mesh ("Head", 6191 vertices) with all 52 ARKit blendshapes as
relative shape keys (plus Basis). Used by the test suite as a facial-animation
asset.

Origin: John Hable's face-scan dataset from the filmicworlds.com article
"Solving Face Scans for ARKit" (source file `colin_dst_rig.fbx`). Capture and
solving by John Hable; talent: Colin Zargarpour. Licensed CC0 (public domain
dedication) -- see `cc0_license.txt` in this directory.

https://filmicworlds.com/blog/solving-face-scans-for-arkit/
